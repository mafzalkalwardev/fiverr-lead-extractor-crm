# v0.2.0 — Client-Ready CRM Release

Fiverr lead research CRM with Next.js dashboard and Python Playwright worker.

## Highlights

- Job queue, MongoDB storage, Excel export
- UTF-8 README fixes and correct clone command
- Compliance section for lawful public-data research

## Install

```powershell
git clone https://github.com/mafzalkalwardev/fiverr-lead-extractor-crm.git
cd fiverr-lead-extractor-crm
npm install
npm run client:start:fast
```

Use demo credentials locally only.
